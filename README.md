# DNDBot

DND Bot for Discord

# Setup

Uses python indicated in `.tool-versons`

## Install discord.py
`pip install discord`

## Install other dependencies (none yet)
`pip install`

## Bot Token
Copy botToken.py.example to botToken.py and add token to use

# Usage

## To Start
`./run`
or run main.py manually

## Commands (prefix is !)

### Everyone
- ping: pong!

### Bot Master
- stop: Stops bot.
